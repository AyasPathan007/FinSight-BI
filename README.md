# 💰 FinSight BI – Financial Performance Dashboard

> **Transforming financial data into executive insights through SQL, Python, and Power BI.**

---

## 📖 Overview

FinSight BI is an end-to-end Business Intelligence project designed to simulate how modern organizations analyze financial performance and support strategic decision-making.

The project demonstrates the complete analytics lifecycle—from designing a relational data model and writing optimized SQL queries to developing interactive Power BI dashboards that provide executives with actionable financial insights.

Rather than focusing solely on visualization, this project emphasizes solving real business problems using data.

---

## 🎯 Business Problem

Organizations generate thousands of financial transactions every day, making it difficult to answer critical business questions such as:

- Are we meeting our revenue targets?
- Which products generate the highest profit?
- Which regions underperform?
- Are we exceeding our budgets?
- How is profitability changing over time?

Traditional spreadsheet reporting often lacks scalability, automation, and interactivity, delaying strategic decision-making.

---

## 💡 Solution

Developed an enterprise-style Business Intelligence solution that:

- Centralizes transactional sales and financial data
- Implements a scalable **Star Schema** data model
- Performs financial analysis using advanced SQL
- Automates KPI reporting
- Delivers interactive executive dashboards in Power BI
- Enables data-driven financial decision-making
