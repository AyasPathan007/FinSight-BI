# KPI Definitions

| KPI | Definition | Formula |
|------|------------|----------|
| Total Sales | Total revenue generated | SUM(Sales Amount) |
| Total Profit | Revenue minus cost | SUM(Profit) |
| Profit Margin % | Profit as a percentage of sales | Profit / Sales × 100 |
| Average Order Value | Average sales per transaction | Sales / Orders |
| Units Sold | Total quantity sold | SUM(Quantity) |
| Customer Count | Distinct purchasing customers | DISTINCT(Customer ID) |
| Return Rate | Percentage of returned orders | Returns / Orders × 100 |
| Inventory Turnover | Inventory utilization | Sales / Average Inventory |
| Budget Variance | Difference between budget and actual | Budget − Actual |
| Regional Growth % | Growth compared with previous period | ((Current - Previous) / Previous) × 100 |
